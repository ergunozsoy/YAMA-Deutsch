# YAMA-Deutsch

**Türkçe konuşanlar için etkileşimli Almanca öğrenme uygulaması**
*Interaktive DaZ-Lernanwendung für türkischsprachige Deutschlernende*

YAMA-Deutsch, Türkçe konuşan Almanca öğrenenlerin en sık yaptığı girişim
(interferenz) hatalarını hedefleyen, karşılaştırmalı dilbilim temelli ücretsiz
bir web uygulamasıdır. Kardeş projesi
[YAMA-DaZ-Sprachvergleich](https://github.com/ergunozsoy/YAMA-DaZ-Sprachvergleich)
ile birlikte bir Digital-Humanities projesi olarak geliştirilmektedir.

## Özellikler

- **Tanı testi:** Öğrencinin hata profilini çıkaran, iki dilli geri bildirim
  veren diagnostik test
- **Alıştırma desenleri:** Tipik Türkçe→Almanca girişim hatalarına odaklanan
  alıştırmalar
- **Cümle denetimi:** Yazılan cümlelerde bilinen hata örüntülerini tanıma
- **Veri odaklı tasarım:** İçerik koddan ayrıdır; alan uzmanları programlama
  bilmeden materyal güncelleyebilir
